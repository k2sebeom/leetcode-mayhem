# Algorithms and Runtime

## Sorting

- Bubble Sort: $O(n^2)$
- Insertion Sort: $O(n^2)$
- Merge Sort: $O(n\log(n))$
- Heap Sort: $O(n\log(n))$

## Heap
- heap push, heap pop: $O(\log(n))$
- heapify: $O(n)$

## Graphs
- Topological Sorting: $O(V+ E)$
- DFS / BFS: $O(V+ E)$
- Dijkstra's algorithm (Shortest Path): $O(E+ V\log(V))$
- Prim's algorithm (MST): $O(V^2)$
